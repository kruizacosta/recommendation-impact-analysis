# Causal Impact of Recommendation Algorithms on User Engagement

## Overview
This project evaluates whether a recommendation algorithm improves user engagement using A/B testing and causal inference methods.

## Methods
- A/B testing (t-test)
- Regression analysis to estimate causal impact

## Key Results
- Statistically significant increase in watch time (p < 0.001)
- Estimated causal lift of ~10 units

## Business Impact
Findings suggest the recommendation algorithm improves user engagement and should be considered for rollout.
